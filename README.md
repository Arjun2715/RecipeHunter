**Setup**

<sup>composer install</sup>

<sup>npm install</sup>

**Set up .env file**

copy env.example an paste as .env 

**Generate app key**

<sup>php artisan generate:key</sup>

**Start the server** 

<sup>php artisan serve</sup>

<sup>npm run dev</sup>
